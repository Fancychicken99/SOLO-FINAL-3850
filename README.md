# SOLO-FINAL-3850
Brody Mensonides

Solo Final for DS3850 

A desktop app for freelancers to track hours and earnings across multiple clients. Logs work sessions, calculates pay, generates invoices, and exports summaries to CSV. Built with Python, tkinter, SQLite, Pandas, and NumPy.

Requires pandas and numpy for use

To Operate Run "pip install pandas numpy" then run "pyton Main.py" in a terminal opened to a directory containing files:
Database.py
Pandas.py
Numpy.py
GUI.py
Invoice.py
